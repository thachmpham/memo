# Investigation Directory
Directory Structure.
```sh
incident_id
            \readme.md      # incident overview
            \timeline.md    # what happen, when
            \theory.md      # theories, proofs
            \log            # logs, traces
            \test.md        # reproduce
            \todo.md        # action points
            ...
```

# Be Detailed. Avoid Quatitative Information.
- If prove a theory, include the error record and the name of the error log file as proof.
